# DnD Helper
### DND Helper is a web application that helps users create, manage, and read Dungeons & Dragons stories and books. Users can write chapters, control book permissions, manage readers, and add audio for chapters.

### Run backend first
### Start Redis
### Install packages

```sh
npm run dev
```

```sh
npm run build
```
